chrome extension that mutes dani's "your mom" jokes and plays "my mom" over them. timestamps are stored in a json i update manually so if i missed one that's on me. it works pretty well but if it doesn't just refresh or something idk.

## how to install

download or clone this repo
go to chrome://extensions
turn on Developer mode (top right)
click Load unpacked
select the folder
go watch a dani video and enjoy
## how it works

when dani says "your mom" in a video, the extension mutes the video and plays a "my mom" audio clip over it. the timestamps for each joke are pulled from a json file so it knows exactly when to do its thing.

## something broken?

refresh the page. if that doesn't work open an issue i guess
